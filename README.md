# Технології збору та обробки даних
## Практичні роботи

Цей репозиторій містить практичні роботи з дисципліни «Технології збору та обробки даних».

Мета курсу — формування навичок роботи з великими наборами даних, оптимізації обробки DataFrame, ефективного читання великих файлів, використання векторизованих операцій та створення власних трансформацій.
